# SuperSonic 项目架构和技术栈文档

## 一、项目概述

**SuperSonic** 是腾讯音乐开发的开源 BI 平台，融合了 **Chat BI**（由 LLM 驱动）和 **Headless BI**（由语义层驱动），旨在打造新一代的 BI 平台。

### 1.1 核心特性

- **Chat BI**：通过自然语言对话进行数据分析
- **Headless BI**：构建统一的语义数据模型
- **Text2SQL**：将自然语言转换为 SQL 查询
- **语义解析**：理解用户查询并生成语义查询语句
- **可视化展示**：自动选择合适的图表呈现数据

## 五、数据库设计

### 5.1 核心表结构

#### 5.1.1 s2_domain（域表）

```sql
CREATE TABLE s2_domain (
    id SERIAL PRIMARY KEY,
    name varchar(100),           -- 域名称
    parent_id integer,            -- 父域 ID
    description TEXT,             -- 域描述
    created_by varchar(100),      -- 创建人
    created_at timestamp,          -- 创建时间
    updated_by varchar(100),      -- 更新人
    updated_at timestamp          -- 更新时间
);
```

**说明**：域是业务划分的最顶层，用于组织相关的模型和数据。

#### 5.1.2 s2_model（模型表）

```sql
CREATE TABLE s2_model (
    id SERIAL PRIMARY KEY,
    domain_id integer,            -- 所属域 ID
    name varchar(100),             -- 模型名称
    description TEXT,              -- 模型描述
    table_name varchar(200),       -- 物理表名
    database_id bigint,           -- 数据库 ID
    type varchar(50),              -- 模型类型
    created_by varchar(100),      -- 创建人
    created_at timestamp,          -- 创建时间
    updated_by varchar(100),      -- 更新人
    updated_at timestamp          -- 更新时间
);
```

**说明**：模型是语义层的核心，定义了数据表的逻辑结构和业务含义。

#### 5.1.3 s2_metric（指标表）

```sql
CREATE TABLE s2_metric (
    id SERIAL PRIMARY KEY,
    model_id bigint,               -- 所属模型 ID
    name varchar(100),             -- 指标名称
    alias varchar(200),            -- 指标别名
    type varchar(50),              -- 指标类型
    expr TEXT,                     -- 指标表达式
    description TEXT,              -- 指标描述
    data_type varchar(50),         -- 数据类型
    created_by varchar(100),      -- 创建人
    created_at timestamp,          -- 创建时间
    updated_by varchar(100),      -- 更新人
    updated_at timestamp          -- 更新时间
);
```

**说明**：指标是业务计算的度量值，如销售额、用户数等。

#### 5.1.4 s2_dimension（维度表）

```sql
CREATE TABLE s2_dimension (
    id SERIAL PRIMARY KEY,
    model_id bigint,               -- 所属模型 ID
    name varchar(100),             -- 维度名称
    alias varchar(200),            -- 维度别名
    type varchar(50),              -- 维度类型
    expr TEXT,                     -- 维度表达式
    data_type varchar(50),         -- 数据类型
    created_by varchar(100),      -- 创建人
    created_at timestamp,          -- 创建时间
    updated_by varchar(100),      -- 更新人
    updated_at timestamp          -- 更新时间
);
```

**说明**：维度是分析数据的角度，如时间、地区、产品等。

## 九、扩展机制

### 9.1 SPI 扩展

SuperSonic 使用 Java SPI 机制实现插件化扩展：

**配置位置**：`META-INF/spring.factories`

**主要扩展点**：
- `SemanticModeller`：语义模型构建

- `SemanticTranslator`：语义翻译器
- `SchemaMapper`：模式映射器
- `ModelProvider`：LLM 模型提供者
